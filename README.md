# 🎮The Data Kingdom: Gamified Alteryx Learning
Transform the way you learn Alteryx designer - from memorizing tools to mastering workflows through a game-like experience. 

# 🌍Overview
**The Data Kingdom** is a **gamified learning framework** designed to help aspiring data analysts and professionals master *Alteryx Designer* through structured levels, challenges and real-world workflows.

Instead of traditional learning, this project introdues:
- Story-driven progression
- Hands-on problem solving
- Real datasets and workflows
- Incremental skill building

# 🎯Why this Project?
Learning tools like *Alteryx Designer* can feel overwhelming when approached theoretically. This project solves that by:
- Breaking down concepts into **progressive levels**
- Turning workflows into **interactive challenges**
- Encouraging **learning by doing**
- Simulating **real-world data problems**

# 🧙‍♀️Your Role: Data Alchemist
You are a Data Alchemist, tasked with restoring balance in a kingdom overwhelmed by chaotic data.

Messy datasets, broken joins, and missing values threaten the system and only your workflows skills can fix them.

# 🗺️Game Structure
## 🌱Level 1: Village of Inputs
**Focus:** Data ingestion & exploration

**Skills:**
- Input Data tool
- Data types understanding
- Data preview

## 🌿Level 2: Forest of Preparation
**Focus:** Data Cleaning

**Skills:**
- Handling null values
- Removing duplicates
- Filtering data

## ⛰️Level 3: Mountains of Transformation
**Focus:** Data manipulation

**Skills:**
- Formula Tool
- Data parsing
- Date transformation

## 🌊 Level 4: River of Joins
**Focus:** Data blending

**Skills:**
- Join types (Inner, Left, Right)
- Union operations

## 🏙️ Level 5: City of Aggregation
**Focus:** Data summarization

**Skills:**
- Grouping
- Aggregations (SUM, AVG, COUNT)

## 🏰 Level 6: Automation Castle
**Focus:** Workflow automation

**Skills:**
- Macros
- Batch processing

## 🔥 Final Boss: The Chaos Engine

**Mission:** Build an end-to-end automated workflow:

```
Input → Clean → Transform → Join → Summarize → Output
```
