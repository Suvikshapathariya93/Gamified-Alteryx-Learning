# 🌱Level 1: Village of Inputs

## 🎯Objective:
Learn how to import, explore and understand raw data in Alteryx designer.

## 🧩Mission: "The Corrupted Scroll"
You have received a raw dataset containing global superstar sales data. Before you can begin your journey, you must understand its structure.

## 📂Dataset
Use this dataset:


(Download and save as CSV)

## ⚔️Tasks
1. Import the dataset using **Input Data Tool**
2. Use **Browse tool** to explore the dataset
3. Identify:
   - Number of rows and columns
   - Column names
   - Data types (String, Int, Date etc.)
4. Use **Select Tool** to:
   - Rename at least 3 columns
   - Change incorrect data types (e.g., Order Date -> Date format)
5. Sort data by **Sales (Descending)**

## 🧠Concepts Covered
- Input Data Tool
- Browse Tool
- Select Tool
- Data Types
- Basic Sorting

## 🏆Expected Outcome
- Clean column names
- Correct data types
- Sorted dataset ready for analysis

## 🎁Bonus Challenge
- Identufy columns with potential data quality issues
- Check for null values manually
